#### 2. 我们如何收集和使用您的个人信息

**2.1 收集目的与范围**  
本服务为实现成绩查询、生成图片等功能，会根据您的操作与选择收集必要的个人信息。我们仅收集实现产品功能所必需的数据，不会超出合理范围。

我们收集的个人信息主要包括以下几类：

- **（1）登录凭证信息：** 当您通过 TapTap 授权或联合查分 API 登录时，我们会接收您提供的 SessionToken、APIID 或平台 ID 等，用于访问您的 Phigros 云存档。该凭证仅用于建立查询会话，不会用于其他目的。
- **（2）操作记录与日志信息：** 当您访问或使用本服务时，服务器会自动记录访问时间、IP 地址、浏览器类型、错误日志等信息，用于服务运行、安全分析与问题排查。
- **（3）生成内容相关数据：** 当您发起成绩查询、生成图片或其他内容时，我们会在服务器内临时处理相应的 Phigros 成绩数据，以便输出结果。

**2.2 数据使用方式**

- **登录凭证：** 仅用于验证授权并访问您的 Phigros 云存档数据。
- **Phigros 云存档数据：** 仅在运行时被解析用于生成查询结果或图片，处理完成后即自动清除，不会存储原始文件。
- **操作记录与日志：** 用于性能监测、安全审计及错误排查，不会与任何个人身份信息关联。

**2.3 第三方统计服务**  
为改进服务质量与了解网站访问情况，本服务可能会使用第三方统计服务（如匿名流量统计或性能分析工具）。  
这些统计服务仅收集与访问行为相关的匿名化信息，例如访问时间、页面停留时长、访问来源等。

此类服务**与 Phigros 云存档及您的登录凭证无任何关联**，不会访问、读取或存储您的游戏数据、SessionToken 或个人标识信息。

**2.4 用户授权与撤回**  
当您授权我们访问您的 Phigros 云存档时，表示您同意我们按本隐私协议所述方式处理您的数据。  
您可随时通过更换 SessionToken、撤销授权或停止使用本服务来撤回授权。  
撤回不影响撤回前基于授权已进行的合法处理活动。
